# Cypress Betslip Automation

## Overview
This project automates the betslip functionality on bet365 using Cypress.

## Test Cases
- Add a bet to the betslip
- Remove a bet from the betslip
- Update the stake amount

## Setup

### Prerequisites

- npm v6.14.13 or later
